# Focus Flow
A simple and efficient desktop To-Do List application built with **Python** and **Tkinter**.

## Features
- Add and delete tasks.
- Clean and simple user interface.
- Saves tasks locally.

## How to Run
1. Install Python.
2. Download **Focus Flow.py**.
3. Run the following command in your terminal:
   ```bash
   python "Focus Flow.py"
   ```
